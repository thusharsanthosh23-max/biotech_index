# Biotech Watchlist Index

An equal-weight 9-stock biotech watchlist index with backtested performance vs XBI, IBB, and SPY.

## Running locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deployed on [Streamlit Cloud](https://share.streamlit.io). Connect this repo and set `app.py` as the entry point.
